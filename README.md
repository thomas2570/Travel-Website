# 🌍 Travel Website

A modern **Tourism Website** built with a premium UI/UX approach.  
Designed to help users explore destinations, discover travel packages, and experience a visually stunning interface.

🔗 **Live Demo:** https://travel-website-sandy-seven.vercel.app/

---

## 🚀 Tech Stack & Architecture

This project uses a high-performance, modern tech stack focused on speed, scalability, and design quality.

### ⚡ Core Framework & Build Tool
- **React** – For building dynamic and interactive UI components  
- **Vite** – Lightning-fast development server and optimized production builds  

### 🎨 Styling & Design
- **Tailwind CSS** – Utility-first CSS framework for responsive layouts and design system  
- **Glassmorphism UI** – Custom blur and transparency effects for a premium look  
- **Dark/Light Mode** – Seamless theme switching  

### ✨ Animations & Icons
- **Framer Motion** – Smooth animations, hover effects, and transitions  
- **Lucide React** – Clean and consistent modern icons  

- Cinematic, ultra-realistic style  
- 4K resolution with golden hour lighting and drone perspectives  
- Consistent visual theme across the website  

### 🌐 Hosting & Version Control
- **Git & GitHub** – Version control and collaboration  
- **Vercel** – Fast deployment with automatic CI/CD and global CDN
